## 🔧 需要手动配置的项目

### 1. 百度站长工具验证（必须）

#### 步骤：
1. 访问 [百度站长平台](https://ziyuan.baidu.com/)
2. 登录并添加你的网站
3. 选择"HTML标签验证"
4. 复制验证代码中的 `content` 值
5. 在 `index.html` 中找到以下行并替换：
   ```html
   <meta name="baidu-site-verification" content="你的验证码">
   ```

#### 示例：
```html
<meta name="baidu-site-verification" content="abc123def456">
```

### 2. 百度自动推送配置（推荐）

自动推送代码已经集成在 `index.html` 中，无需额外配置。

**验证是否生效：**
- 在百度站长工具中查看"数据引入" -> "链接提交" -> "自动推送"
- 应该能看到推送的URL记录

### 3. 百度API推送（可选，更高效）

如果需要更高效的推送方式，可以使用API推送：

1. 在百度站长工具中获取：
   - 站点（site）
   - Token

2. 修改 `src/utils/seo.ts` 中的 `pushToBaidu` 函数：
   ```typescript
   const apiUrl = 'https://data.zz.baidu.com/urls?site=你的站点&token=你的token'
   ```

3. 取消注释API推送代码

### 4. 百度统计代码（可选）

如果需要使用百度统计：

1. 访问 [百度统计](https://tongji.baidu.com/)
2. 添加网站并获取统计代码
3. 在 `index.html` 的 `</head>` 标签前添加：
   ```html
   <script>
     var _hmt = _hmt || [];
     (function() {
       var hm = document.createElement("script");
       hm.src = "https://hm.baidu.com/hm.js?你的统计ID";
       var s = document.getElementsByTagName("script")[0];
       s.parentNode.insertBefore(hm, s);
     })();
   </script>
   ```

### 5. 提交Sitemap到百度

1. 登录百度站长工具
2. 进入"数据引入" -> "链接提交" -> "Sitemap"
3. 提交你的sitemap地址：`https://szw.lmzy98.cn/sitemap.xml`

### 6. 移动适配配置

如果你的网站有独立的移动端，需要在百度站长工具中配置：

1. 进入"移动适配"
2. 添加PC和移动端URL对应关系
3. 或使用自适应设计（当前网站已使用）
